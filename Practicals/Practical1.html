About Course

Full Stack development refers to the development of both Front-end(client side) and Back-end(server side). It includes various languages as well as various frameworks for front-end and back-end. Typically languages used in this course are Hypertext Markup Language(HTML), Cascading style sheets (CSS) and Javascript. HTML is used to develop static webpages basically it is used to display the content on the webpage, CSS is used to style the content on the webpage and Javascript gives the behaviour to the webpage i.e. make it responsive. We are having various frameworks like react, bootstrap etc. for front-end whereas for back-end, we can use nodejs, Django etc. Languages that can be used for back-end are java, ruby, python etc. Overall the main objective of this course is to develop a website that are responsive, creative and well defined.
